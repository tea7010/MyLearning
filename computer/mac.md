# ショートカット忘備録
* アプリ等検索: `Command + Space`
* Copy/Cut/Paste: `Command + C/X/V`
* Undo/Redo: `Command + Z`, `Command + Shift + Z`
* Save: `Command + S`
* VScodeのコマンドパレット: `Command + Shift + P`

# 気付いたこと・対処
