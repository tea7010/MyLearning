# MD770/MD650のカスタムキーマップ

###### tags: `computer`

## MD770

### 基本操作

#### レイヤーの切り替え
* Pn + {m, <, >, ?}: レイヤー0, １, 2, 3

#### キーマクロアサイン
* Fn + R-Ctrl -> ( {Assign key} -> {Input} -> Pn ) -> Fn + R-Ctrl
* キーを入れ替えたいときは各キーで上を相互に実施
* Pnは変更不可能
* Fnの入れ替え: Fn or Pn + Left-Shiftを3秒長押し -> 白が点滅するので、割り当てたいキーを入力

### レイヤー2(Green)
* Caps -> Ctrl
* R-Space -> Fn
* Fn -> Windows
* Home -> Ctrl + Shift + M(Teams, ミュートの切り替え)
* PgUp -> Ctrl + Shift + K(Teams, 手をあげる、さげる)
```
Fn(右親指) + 

2|3|4|5
-|-|-|-
W|E|R|T
S|D|F|G
X|C|V|B

-> 

2|3|4|5
-|-|-|-
1|2|3|T
4|5|6|G
7|8|9|0
```

### レイヤー3(青)
* Caps -> Ctrl
* Backspace <-> \|
* R-Alt -> Windows
* Esc <-> `~
* Pgup -> `~
* R-Space -> Fn
