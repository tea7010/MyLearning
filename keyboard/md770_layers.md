# MD770/MD650のカスタムキーマップ

###### tags: `computer`

## MD770

### メモ
* Pnは変更不可能
* Pn + m: デフォルトレイヤー
* Pn + <, >, ?: レイヤー１, 2, 3
* Fn + R-Ctrl -> ( {Assign key} -> {Input} -> Pn ) -> Fn + R-Ctrl
* キーを入れ替えたいときは各キーで上を相互に実施
* Fnの入れ替え: Fn or Pn + Left-Shiftを3秒長押し -> 白が点滅するので、割り当てたいキーを入力

### レイヤー2(Green)
* Caps -> Ctrl
* R-Space -> Fn

### レイヤー3(青)
* Caps -> Ctrl
* Backspace <-> \|
* R-Alt -> Windows
* Esc <-> `~
* Pgup -> `~
* R-Space -> Fn


## MD650
* Fnの入れ替え: Fn or Pn + Left-Shiftを3秒長押し -> 赤が点滅するので、割り当てたいキーを入力

### レイヤー2(Green)
* Caps -> Ctrl
* R-Space -> Fn

### レイヤー3(青)
* Caps -> Ctrl
* Backspace <-> \|
* R-Space -> Fn
* del -> `~
* PGUP -> del
