* .: Any
* \: escape 
    * e.g. `\.`
* \d: digit number
* \D: Non-digit number
* []: only 
    * [abc]: Only a, b, or c
    * [^abc]: Not a, b, nor c
    * [a-z]: a to z(lower)
    * [A-Z]: A to Z(uppper)
    * [0-9]: 0 to 9
    * \w: Any alphanumeric characters = [A-Za-z0-9_]
    * \W: Non-Any alphanumeric
* {}: repeat
* *: zero or more
* +: one or more
